# Cognifyz Task 5 - API Integration and Front-End Interaction

**Internship Program:** Cognifyz Full Stack Development
**Level:** 3 - Advanced
**Task:** API Integration and Front-End Interaction

## Objective
Introduce server-client communication through a RESTful API with full CRUD operations.

## Features
- RESTful API endpoints (GET, POST, PUT, DELETE)
- Front-end interface that interacts with the server API
- Fetch and display data from the API dynamically
- In-memory data storage on the server
- CORS enabled for cross-origin requests
- Responsive UI matching Ethereal Mono design system

## Tech Stack
- Node.js
- Express.js
- Vanilla JavaScript (Fetch API)
- Tailwind CSS (via CDN)
- Google Fonts (Hanken Grotesk, Inter, JetBrains Mono)
- Material Symbols Icons
- GitHub Actions (CI)

## API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /api/users | Get all users |
| GET | /api/users/:id | Get user by ID |
| POST | /api/users | Create new user |
| PUT | /api/users/:id | Update user |
| DELETE | /api/users/:id | Delete user |

## How to Run Locally
npm install
node server.js
Then open http://localhost:3000 in your browser.
